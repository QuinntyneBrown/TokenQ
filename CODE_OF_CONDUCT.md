# Code of Conduct

TokenQ is intended to be a professional, respectful open source project.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent, but accept correction when impact differs from intent.
- Keep discussions focused on the project and its users.
- Give clear technical feedback without personal attacks.
- Respect maintainers' decisions about project scope and priorities.

## Unacceptable Behavior

- Harassment, threats, insults, or discriminatory language.
- Sexualized language or imagery.
- Publishing private information without explicit permission.
- Sustained disruption of issues, pull requests, or discussions.
- Any behavior that would reasonably make contributors feel unsafe or unwelcome.

## Enforcement

Maintainers may remove comments, close issues, reject contributions, or block
participants who violate this code. Serious incidents should be reported
privately through the maintainer contact path listed in [SUPPORT.md](SUPPORT.md).

Enforcement decisions should be proportional, documented privately when needed,
and focused on preserving a healthy project environment.
